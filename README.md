# Boda Alejandro & Elvira

Wedding website built with React + Vite + Tailwind CSS.

## Setup

```bash
git clone https://github.com/kodemonk/boda.git
cd boda
npm install
npm run dev
```

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |

## Deployment

Hosted on Netlify with auto-deploy. Push to `main` triggers automatic build and deployment.

## Requirements

- Node.js
- GitHub access

No additional credentials, environment variables, or API keys needed.
